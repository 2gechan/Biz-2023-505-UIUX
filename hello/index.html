처음으로 만드는 Web Page

처음으로 만드는 HTML 문서로 작성된 Web Page, Web Page 는 HTML 형태의 문서로 작성한다. HTML 문서에서 p tag 로 시작하면 하나의 문단이 시작된다. 하나의 문단은 close /p 이며 문단이 끝났음을 나타낸다. 문단 내에서는 아무리 Enter 를 입력하여도 Browser 에 나타날 때는 모두 무시된다. 단 Enter 를 입력하면 한 칸의 빈 칸이 나타난다. 문단 중간에서 여러 개의 빈 칸(space)을 입력하여도 한 개의 빈 칸만 남기고 모두 무시된다.

HTML 문서는 tag 로 감싸진 Text 로 구성되어 있다. tag 로 감싸진 문서를 작성하고 Web Browser 를 통하여 확인한다
